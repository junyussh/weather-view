所需技能

團隊使用 Git 進行版本控制，所以欲加入前請先學會
Git 版本控制

前端工程師

  1. 有開發 HTML5/CSS3/Javascript 的經驗
  2. 熟悉使用 SASS/SCSS 語法糖開發 CSS
  3. 能手刻網頁、RWD，並利用 Susy2 進行排版
  4. 使用 Gulp/Webpack 進行任務自動化管理
  5. 主動嘗試新技術

其他條件：

  1. UI/UX 設計
  2. 會使用前端框架如 React.js/Vue.js

後端工程師

  1. 熟悉 Python3
  2. 熟悉Django Web Framework
  3. 熟悉Linux操作
  4. 熟悉SQL指令
  5. 能使用基本HTML/CSS/javascript
  6. 有使用Nginx伺服器部署動態網站的經驗

其他條件：

  1. 有Scikit-learn開發經驗
  2. 能善用stackoverflow、具備解決問題的能力

硬體工程師

  1. 有開發 Arduino/8051 的經驗
  2. 熟悉 Python 與 C/C++
  3. 熟悉基本電學
  4. 需會使用 Linux 系統並熟悉 CLI 介面
  5. 會焊電路板

其他條件：

  1. 接觸過組合語言，有記憶體基礎
  2. 隨時更新知識
